Burp Suite Lab
هذا المستودع مخصص لإعداد بيئة عمل (Lab) لاختبار الاختراق باستخدام Burp Suite وتطبيق OWASP Juice Shop. هذا المختبر هو جزء من المحتوى التعليمي المقدم على قناة Qais Cyber.

🛠️ المتطلبات الأساسية
تأكد من وجود البرامج التالية على جهازك:

Node.js

Burp Suite

Git (لتحميل المستودع)

🚀 خطوات التحميل والتشغيل
اتبع الأوامر التالية بالترتيب لتشغيل المختبر:

1. تحميل المشروع (Clone)
افتح واجهة السطر البرمجي (Terminal) وقم بنسخ المستودع:
```
Bash
git clone https://github.com/qaiscyber/Burp-suite-Lab.git
```
2. التثبيت والدخول إلى مجلد المشروع 
```
Bash
bash setup.sh
cd juice-shop_14.0.1
```
3. تشغيل التطبيق
```
Bash
npm start
```
🔍 كيفية الاستخدام
بعد التشغيل، توجه إلى الرابط: http://localhost:3000.

قم بضبط Burp Suite Proxy لاعتراض الطلبات.

ابدأ بتطبيق تقنيات اختبار الاختراق التي تعلمتها في الكورس.
